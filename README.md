# MDEClust
MDEClust provides both parallel and non-parallel implementations of the Memetic Differential Evolution (MDE) algorithm for solving the Euclidean Minimum Sum-of-Squares Clustering (MSSC) problem.

Programmed by Rustam Mussabayev (rmusab@gmail.com)
Initial release: 20 August 2022
